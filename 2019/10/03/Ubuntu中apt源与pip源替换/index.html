Ubuntu中apt源与pip源替换

发表于 2019-10-03 分类于 Linux

Advanced Packaging Tool (apt) 是Linux下的安装包管理工具。Pip 是 Python 包管理工具,该工具提供了对Python 包的查找、下载、安装、卸载的功能。更换源的必要性在于提高各种包的下载速度。

Apt源更换为阿里镜像源

进入ubuntu“软件与更新”。ubuntu软件分页中，“下载自”选项选择使用http://mirros.aliyun.com/ubuntu 阿里云镜像源。
